# Romania Cybersecurity Intelligence Report 2026

Static GitHub Pages site for the Romania cyber intelligence report.

## Local files

- `index.html`: GitHub Pages entrypoint
- `romania_cyber_report_2026.html`: main report page
- `.github/workflows/pages.yml`: automatic deployment workflow

## Publishing

1. Push this repository to the `main` branch of `https://github.com/danlex/ro-cyber-intel-2026`.
2. In GitHub, open `Settings -> Pages`.
3. Set `Source` to `GitHub Actions`.
4. The `Deploy GitHub Pages` workflow will publish the site on each push to `main`.
